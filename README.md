# Тестирование REST API JSONPlaceholder

## Что это?
Набор ручных тест-кейсов и готовая коллекция запросов в Postman для проверки функциональности публичного REST API [JSONPlaceholder](https://jsonplaceholder.typicode.com).

## Цель
Продемонстрировать навыки написания тестовой документации и работы с Postman.

## Инструменты
- **Postman** – составление запросов, переменные окружения, запуск коллекций.
- **JSONPlaceholder** – тестовый REST API (open source).
- **GitHub** – хранение документации и отчёта.

## Содержимое репозитория
- `README.md` – описание проекта.
- `test-cases/` – тест-кейсы для основных ресурсов API (/posts, /comments, /albums, /photos, /todos, /users).
- `test-report.md` – краткий отчёт о тестировании со ссылкой на коллекцию Postman и скриншотами прогонов.
- `collection_run_1.png` | `collection_run_2.png` - Скриншоты прогона коллекции
